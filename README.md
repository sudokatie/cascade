# Cascade

Time-loop survival. Death resets the day, but your changes persist. Dig a tunnel on loop 3, it exists on loop 4. Die to a trap, leave a warning for your next self. Break the loop or perfect it.

## Why This Exists?

Most survival games punish death by taking your stuff. Cascade punishes death by making you relive the day — but smarter. Knowledge persists. Structures persist. The world resets, you don't. It's Groundhog Day with pickaxes.

## Features

- Time loop mechanics with 4 phases (Dawn, Day, Dusk, Midnight)
- Persistent state across loops (terrain, structures, temporal chests)
- Paradox system from conflicting state changes
- Knowledge system (discovered traps, recipes, routes survive death)
- Loop-aware creatures that adapt to your patterns
- Chrono Boots that stack speed across loops
- 3 trap types (static, loop-persistent, paradox hazards)
- Temporal chests for cross-loop item storage

## Quick Start

```bash
cargo build
cargo run
```

Built on the [Lattice](https://github.com/sudokatie/lattice) survival engine.
